## 制作サイトのタイトル

キッズもニッコリ じゃんけんアプリ！

## 制作サイトの説明（40 文字程度）

デザインを優しめにして、子ども向けじゃんけんアプリを作成しました。

## 工夫した点・こだわった点

相手の手を高速ランダムで動かすことで、目押しされないようにしました。  
画面録画だと聞こえませんが、勝った時のみ音がなるようになっています。

## 難しかった点・次回トライしたこと（または機能）

ボタンをクリックした時に、テキストやボタンを表示させるのが大変でした。

## 備考（感想、シェアしたいこと等なんでも）

youtube で初心者向けのクイズゲームを作成する動画を見つけたので、勉強を兼ねて観ながら一緒に作ってみました。  
動画を観ながら一緒に作ったので、基本システムは完全に動画そのまま書きました…！！供養として共有させてください。

一箇所、動画だとアラートで結果表示だったところを、じゃんけんアプリと同じようにテキストで表示されるようにしてみました。  
じゃんけんアプリが子ども向けイメージで作ったので、こっちは大人っぽい（？）色味にしてみました。
